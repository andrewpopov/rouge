# Meta Builds — Optimal Endgame Decks

Each class has 2-3 viable builds. For each build: the synergy loop, ideal deck composition, target deck size, and key progression milestones.

## Barbarian

### Berserker (combat_skills / martial)
**Loop**: Stack attack windows → chain big hits
**Target deck**: 14-16 cards
**Core (8)**: bash, double_swing, concentrate, frenzy, axe_mastery, stun, weapon_mastery, battle_charge
**Support (4-6)**: iron_skin, find_potion, war_stance, natural_resistance
**Capstone**: berserk, combat_mastery (aura), fury_mastery
**Why it works**: Axe mastery (+3 next attack) and increased_stamina (+3 next 2) stack into bash (14) and frenzy (10+9). Tight deck means you see your combo every 2-3 turns.

### Warcrier (warcries / command)
**Loop**: Debuff enemies → buff party → sustained attrition
**Target deck**: 16-18 cards
**Core (8)**: howl, shout, battle_cry, battle_orders, war_cry, fury_howl, warlord_shout, find_potion
**Support (6)**: iron_skin, natural_resistance, sword_mastery, bash, increased_stamina, bulwark
**Capstone**: battle_command (aura), ancient_vow, unyielding
**Why it works**: Warcry cards generate guard + draw + debuff. Battle_command aura gives all warcries +1 draw +4 guard. Turns into a value engine that out-sustains any boss.

## Druid

### Summoner (summoning / command)
**Loop**: Build minion board → scale summon damage → pack synergy payoffs
**Target deck**: 16-20 cards
**Core (8-10)**: raven, poison_creeper, oak_sage, spirit_wolf, dire_wolf, heart_of_wolverine, pack_howl, natures_wrath
**Support (6-8)**: cyclone_armor, lycanthropy, firestorm, werebear, natures_balance, renewal
**Capstone**: summon_grizzly, wild_stampede, force_of_nature
**Why it works**: Each summon adds persistent DPS (4-10/turn). Pack_howl gives all summons +4 damage. At 3+ summons, wild_stampede deals 15-20 AoE. Draw from natures_balance (if summon) keeps the engine running.

### Elementalist (elemental / arcane)
**Loop**: Apply burn/slow → trigger status payoffs → AoE burst
**Target deck**: 14-16 cards
**Core (7)**: firestorm, molten_boulder, fissure, volcano, tornado, gale_force, eruption
**Support (5-7)**: cyclone_armor, lycanthropy, werebear, arctic_blast, twister
**Capstone**: armageddon, hurricane, wild_convergence, elemental_mastery (aura)
**Why it works**: Firestorm applies burn, twister/arctic_blast apply slow. Tornado deals +4 to all slowed enemies. Eruption deals +4 per burn stack. Elemental_mastery aura makes all burn +2 and slow +1 turn. Tight deck = consistent combos.

### Shapeshifter (shape_shifting / martial)
**Loop**: Melee multi-hit + sustain → outlast via lifesteal
**Target deck**: 12-14 cards
**Core (6)**: werewolf, werebear, feral_rage, fury, fire_claws, maul
**Support (4-6)**: cyclone_armor, lycanthropy, natures_balance, rabies
**Capstone**: hunger, primal_fury, savage_pounce
**Why it works**: Werewolf (11 dmg + heal 5), fury (12+11+10 = 33 total), hunger (+6 if slowed). Feral_rage buffs next melee +3. Smallest deck = fastest cycle = most damage per turn. Pure aggro.

## Necromancer

### Summoner (summoning / command)
**Loop**: Build skeleton army → scale with summon count → corpse cycle
**Target deck**: 16-20 cards
**Core (8)**: raise_skeleton x2, clay_golem, skeletal_mage, blood_golem, skeleton_mastery, golem_mastery, bone_offering
**Support (6)**: life_tap, bone_armor, amplify_damage, dark_pact, soul_harvest
**Capstone**: army_of_dead, corpse_harvest, revive
**Why it works**: Skeleton army deals 7+ per minion per turn. Skeleton_mastery gives all summons +4. Army_of_dead scales 5 dmg per summon. Soul_harvest draws if you have summons. Amplify_damage marks enemies for merc +8.

### Bone Mage (poison_and_bone / arcane)
**Loop**: Apply poison → trigger bone payoffs → corpse explosion chains
**Target deck**: 14-16 cards
**Core (7)**: teeth, bone_spear, poison_dagger, corpse_explosion, poison_explosion, poison_nova, bone_wall
**Support (5-7)**: bone_armor, life_tap, amplify_damage, lower_resist, dark_pact
**Capstone**: bone_spirit, corpse_lance, bone_plague (aura)
**Why it works**: Poison_dagger and poison_explosion spread poison. Bone_spear deals +6 if target has poison. Corpse_explosion chains +6 AoE when enemies die. Lower_resist amplifies next 2 poison applications by +3. Kill one enemy → corpse explosion → kill more → cascade.

## Paladin

### Guardian (defensive_auras / support)
**Loop**: Aura stacking → party sustain → mercenary-carried damage
**Target deck**: 16-18 cards
**Core (8)**: prayer, defiance, holy_freeze, cleansing, meditation, holy_shield, salvation, righteous_guard
**Support (6)**: smite, might, holy_bolt, sacrifice, thorns
**Capstone**: redemption, divine_renewal, blessed_ward, aura_mastery (aura)
**Why it works**: Prayer heals party. Defiance gives party guard + taunt. Holy_freeze does 8 AoE + freeze. Aura_mastery makes all aura cards +4 damage +4 guard. Merc does the killing while paladin sustains.

### Zealot (offensive_auras / support)
**Loop**: Apply burn/slow → buff attacks → holy payoff cascade
**Target deck**: 14-16 cards
**Core (7)**: holy_fire, thorns, charge, blessed_aim, vengeance, holy_strike, concentration
**Support (5-7)**: prayer, cleansing, defiance, smite, might
**Capstone**: crusade, righteous_wrath, divine_command, fanaticism
**Why it works**: Holy_fire applies burn 4. Thorns applies burn 2. Then holy_strike deals +9 if target has burn/slow. Blessed_aim buffs next 2 attacks +4. Crusade gives party +5 damage. Concentration buffs next aura/attack +5. Everything chains into massive burst turns.

## Assassin

### Shadow Master (shadow_disciplines / command)
**Loop**: Cost reduction → rapid card chains → draw acceleration
**Target deck**: 14-16 cards
**Core (7)**: psychic_hammer, burst_of_speed, cloak_of_shadows, blade_shield, shadow_shroud, shadow_veil, cobra_strike
**Support (5-7)**: claw_mastery, fire_blast, weapon_block, tiger_strike
**Capstone**: lethal_tempo (aura), shadow_storm, natalyas_guard
**Why it works**: Shadow_shroud (-1 cost next Assassin) and shadow_veil (-1 cost next 2) enable rapid card chains. Lethal_tempo aura gives all Assassin cards +1 draw when they draw. Card velocity snowballs.

### Martial Artist (martial_arts / martial)
**Loop**: Setup melee buffs → combo finishers → dragon payoffs
**Target deck**: 12-14 cards
**Core (6)**: tiger_strike, cobra_strike, fists_of_fire, dragon_talon, dragon_claw, venom
**Support (4-6)**: claw_mastery, blade_shield, burst_of_speed, weapon_block
**Capstone**: dragon_flight, dragon_tail, fatal_cascade, death_blossom
**Why it works**: Tiger_strike buffs next melee +3. Venom buffs next melee +4. Then dragon_flight deals +10 if played melee earlier. Dragon_tail deals 5 AoE if played melee. Death_blossom deals +12 if target has burn/poison/paralyze. Ultra-tight deck = combo fires every turn.

### Trapper (traps / arcane)
**Loop**: Deploy persistent trap damage → buff trap power → free hero to guard
**Target deck**: 14-16 cards
**Core (6)**: fire_blast, wake_of_fire, shock_web, blade_sentinel, lightning_sentry, trap_mastery
**Support (6-8)**: burst_of_speed, cloak_of_shadows, blade_shield, psychic_hammer, weapon_block
**Capstone**: death_sentry, shadow_storm (+4 trap damage)
**Why it works**: Traps persist for 3 turns, dealing damage every turn without hero action. Shadow_storm gives all traps +4 damage. While traps do damage, hero plays guard/draw cards. Board control build.

## Sorceress

### Lightning (lightning / arcane)
**Loop**: Spell chains → paralyze control → escalating AoE
**Target deck**: 14-16 cards
**Core (7)**: charged_bolt, static_field, chain_lightning, lightning, nova, thunder_storm, spell_surge
**Support (5-7)**: energy_shield, warmth, enchant, arcane_focus
**Capstone**: lightning_mastery, overcharge, arc_mastery (aura)
**Why it works**: Thunder_storm buffs next spell +5. Spell_surge buffs next spell +6. Then chain_lightning or nova deals massive AoE + paralyze. Overcharge deals +6 AoE if played spell earlier. Arc_mastery gives all lightning spells +3 damage +1 draw. Draw engine + damage engine in one.

### Fire (fire / arcane)
**Loop**: Burn stacking → burn payoff explosion
**Target deck**: 14-16 cards
**Core (7)**: fire_bolt, fireball, inferno, blaze, fire_wall, meteor, combustion
**Support (5-7)**: warmth, energy_shield, enchant, arcane_focus
**Capstone**: hydra, conflagration, fire_mastery (aura)
**Why it works**: Fire_bolt/fireball/inferno stack burn. Combustion deals 3x damage per burn stack (max 5). Conflagration deals +8 to enemies with 4+ burn. Fire_mastery aura gives all fire spells +3 damage +2 burn. Burn becomes the primary damage source.

### Frost (cold / arcane)
**Loop**: Freeze/slow control → status payoff → defensive tempo
**Target deck**: 14-16 cards
**Core (7)**: ice_bolt, frost_nova, blizzard, glacial_spike, ice_blast, frost_bolt, frozen_orb
**Support (5-7)**: energy_shield, warmth, teleport, shiver_armor
**Capstone**: cold_mastery (aura), chilling_armor, tempest
**Why it works**: Ice_bolt/frost_nova apply slow/freeze. Ice_blast deals +6 if target already slowed. Frost_bolt converts slow to freeze if fire spell played. Cold_mastery gives cold spells +3 damage +1 slow. Enemies are permanently crowd-controlled.

## Amazon

### Marksman (bow_and_crossbow / martial)
**Loop**: Apply slow → mercenary marks → ranged burst
**Target deck**: 14-16 cards
**Core (7)**: magic_arrow, cold_arrow, guided_arrow, multiple_shot, strafe, inner_sight, penetrate
**Support (5-7)**: dodge, critical_strike, inner_calm, exploding_arrow
**Capstone**: pierce, freezing_arrow, arrow_mastery (aura), thunder_volley
**Why it works**: Inner_sight applies slow + marks for merc +8. Cold_arrow applies slow. Then precision deals +5 if slowed + draws 1. Arrow_mastery aura gives ranged cards +3. Guided_arrow (16 dmg) and pierce (26 dmg) are finishers. Merc adds 8-18 damage per marked hit.

### Javazon (javelin_and_spear / martial)
**Loop**: Apply paralyze → trigger javelin payoffs → AoE lightning burst
**Target deck**: 14-16 cards
**Core (7)**: jab, power_strike, charged_strike, lightning_bolt, lightning_strike, javelin_rush, poison_javelin
**Support (5-7)**: dodge, inner_sight, critical_strike, decoy
**Capstone**: lightning_fury, storm_javelin, javelin_mastery (aura), war_pike
**Why it works**: Power_strike (11 + paralyze 2) and charged_strike (12x2 + paralyze 3) apply paralyze. Storm_javelin deals +8 if target paralyzed. Lightning_fury deals 13 AoE + paralyze 3. Javelin_mastery aura gives javelin cards +3 damage +1 paralyze. Chain reaction of paralyze → payoff → more paralyze.

## Build Size Summary

| Archetype Type | Target Size | Reasoning |
|---|---|---|
| Martial Combo (Barb Berserker, Assassin Martial, Druid Shapeshifter) | 12-14 | Need to see whole deck every 2 turns for combos |
| Spell Chain (Sorceress all, Assassin Shadow) | 14-16 | Need draw density + spell chain reliability |
| Engine Build (Summoner, Warcrier, Guardian) | 16-20 | More setup cards, board-building over time |
| Status Payoff (Zealot, Elementalist, Bone Mage, Amazon both) | 14-16 | Apply status then payoff, needs consistency |
