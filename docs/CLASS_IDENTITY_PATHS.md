# Class Identity Paths

Last updated: March 29, 2026.

## Purpose

This document defines the intended build-identity paths for each class at the roster level.

Use it for:

- class-level balance direction
- deciding which lanes are flagship versus secondary
- checking whether a named lane deserves full committed-suite validation
- keeping class strategy docs aligned with the actual target fantasy

This is a target-design doc, not a claim that every lane is already equally healthy in the live build.

## Design Rules

- Every class should have `3` named identity paths.
- Every named path should answer:
  - what the deck is trying to do
  - what early choices commit the run
  - what weapon family or gear line supports it
  - what boss problem it must solve
- Every class should have at least `1` flagship path that feels like the clearest expression of the class fantasy.
- Secondary paths do not need equal power, but they must be real build destinations rather than flavor-only branches.
- If a path is named here, it should either have a committed balance suite or be explicitly downgraded later.

## Amazon

### Bow Volley

- Role: flagship path
- Core fantasy: precision ranged pressure with stable draw quality
- What it should feel like: the cleanest “I drafted a plan and it keeps working” Amazon run
- Early commitment signals: bow picks, passive support, ranged payoff turns
- Gear direction: bows or crossbows, runes or runewords that reinforce repeated ranged turns
- Boss ask: survive long enough to line up several payoff cycles instead of trying to one-turn the fight

### Javelin Storm

- Role: secondary high-upside path
- Core fantasy: tempo spikes, shock pressure, and committed attack windows
- What it should feel like: explosive and dangerous, with real punish if the run overextends
- Early commitment signals: javelin or spear progression, burst-heavy attack rewards
- Gear direction: javelins first, spears as fallback
- Boss ask: build enough guard or tempo protection that burst windows are not suicide turns

### Passive Tempo

- Role: secondary flexible path
- Core fantasy: efficiency, dodge, crit-style scaling, and mercenary-assisted tempo
- What it should feel like: smart, low-waste combat that still needs a real finisher
- Early commitment signals: passive-heavy early rewards without a locked weapon lane
- Gear direction: weapon line stays open longer, but should still resolve into bow or javelin support
- Boss ask: convert efficiency into an actual closing plan before late bosses stabilize

## Assassin

### Martial Burst

- Role: flagship path
- Core fantasy: set up, hold nerve, then cash in one or two decisive burst turns
- What it should feel like: the most sequencing-sensitive melee deck in the roster
- Early commitment signals: martial picks, setup-plus-payoff rewards, aggressive sword progression
- Gear direction: swords first, claw-like fantasy can be represented later if the item system expands
- Boss ask: survive long enough to convert the setup turn into a real swing turn

### Trap Field

- Role: secondary full-run path
- Core fantasy: delayed area control and encounter-shape manipulation
- What it should feel like: planning ahead, surviving awkward hands, then letting the field take over
- Early commitment signals: repeated trap rewards, setup cards, shadow support that protects slower turns
- Gear direction: sword as default live lane, caster fallback only as a temporary stopgap
- Boss ask: become a real late-game control strategy, not just an Act I-II gimmick

### Shadow Tempo

- Role: secondary flexible path
- Core fantasy: evasion, clones, adaptation, and medium-value tactical play
- What it should feel like: the lane with the most “I had options” turns
- Early commitment signals: shadow skills, evasion support, tempo smoothing rewards
- Gear direction: sword path with support-heavy itemization
- Boss ask: prove it can still close fights instead of playing fair forever

## Barbarian

### Combat Pressure

- Role: flagship path
- Core fantasy: proactive frontline damage with enough defense to keep swinging
- What it should feel like: the clearest “barbarian attacks, but intelligently” lane
- Early commitment signals: combat tree picks, strong early melee weapon progression
- Gear direction: swords and maces first, with axes as future expansion room
- Boss ask: stop being a pure race deck and learn to live through telegraphed punishment

### Mastery Frontline

- Role: secondary stable path
- Core fantasy: weapon-specialist scaling and fewer dead turns
- What it should feel like: the most reliable Barbarian lane, with better tolerance for bad draws
- Early commitment signals: mastery picks plus a stable weapon line
- Gear direction: whichever melee family is actually advancing well, preferably sword or mace
- Boss ask: keep enough top-end payoff that “stable” does not turn into “toothless”

### Warcry Tempo

- Role: secondary synergy path
- Core fantasy: buffs, taunts, guard, mercenary timing, and turn manipulation
- What it should feel like: the most deckbuilder-shaped Barbarian plan
- Early commitment signals: warcry density, guard support, mercenary scaling
- Gear direction: mace-biased frontline weapons
- Boss ask: avoid low-damage stall by turning support windows into real kill turns

## Druid

### Elemental Storm

- Role: flagship path
- Core fantasy: delayed spell payoff and typed-damage conversion
- What it should feel like: fragile early, powerful once the engine turns on
- Early commitment signals: elemental picks, staff progression, spell-payoff rewards
- Gear direction: staves first
- Boss ask: survive the early cycles, then convert spell windows into a real finish

### Shifter Bruiser

- Role: secondary stable path
- Core fantasy: durable melee chaining with fewer awkward dead turns
- What it should feel like: the most honest and sturdy Druid path
- Early commitment signals: shape-shifting picks plus mace or bruiser weapon upgrades
- Gear direction: maces first
- Boss ask: keep enough reach that it does not stall on late bosses

### Summoner Engine

- Role: secondary engine path
- Core fantasy: stabilize the board, then win through scaling presence
- What it should feel like: the clearest “engine deck” inside Druid
- Early commitment signals: summon density, board-support rewards, slower engine value
- Gear direction: staves first, mace fallback if the live run fails to find caster support
- Boss ask: survive the vulnerable setup window and reach a board state that actually matters

## Necromancer

### Bone Burst

- Role: flagship path
- Core fantasy: control the fight, then detonate with bone payoff
- What it should feel like: a deliberate setup-and-kill caster deck, not generic mage value
- Early commitment signals: bone payoff, curse support, wand upgrades
- Gear direction: wands first, with runewords that clearly reinforce bone scaling
- Boss ask: line up a real finishing sequence before the fight overwhelms the setup

### Curse Control

- Role: secondary tactical path
- Core fantasy: weaken enemies, buy time, and convert attrition into wins
- What it should feel like: careful, oppressive, but never passive for its own sake
- Early commitment signals: curse density, long-fight support, survivability picks
- Gear direction: wands with utility-forward scaling
- Boss ask: prove that control becomes a kill plan, not just a delay

### Summon Swarm

- Role: secondary engine path
- Core fantasy: scale around minion presence and support sequencing
- What it should feel like: the most board-centric Necromancer lane
- Early commitment signals: summon density, engine support, minion-protection cards
- Gear direction: wand path that supports summon value rather than replacing it
- Boss ask: make it through the setup window and avoid passive dead-end decks

## Paladin

### Combat Zeal

- Role: flagship path
- Core fantasy: repeated melee attacks backed by just enough aura structure
- What it should feel like: intuitive, righteous, and safer than it first appears
- Early commitment signals: combat skills, attack cadence, auras that reward staying on offense
- Gear direction: sword or mace
- Boss ask: respect burst turns instead of mindlessly swinging through them

### Offensive Aura

- Role: secondary synergy path
- Core fantasy: aura scaling that turns ordinary attacks into major turns
- What it should feel like: the most build-around Paladin lane
- Early commitment signals: offensive aura density plus enough attack cards to matter
- Gear direction: sword or mace, biased toward whichever line gets the better on-hit payoff
- Boss ask: assemble the engine early enough that the boss does not outscale the setup

### Defensive Anchor

- Role: secondary attrition path
- Core fantasy: long-fight control through survivability and stability
- What it should feel like: safe, but only if drafted with a real closing plan
- Early commitment signals: defensive aura value, sustain picks, stable frontline gear
- Gear direction: mace-biased frontline gear
- Boss ask: turn survival into inevitability instead of merely delaying defeat

## Sorceress

### Fire Burst

- Role: flagship path
- Core fantasy: explosive payoff turns after a vulnerable setup window
- What it should feel like: the classic high-power Sorceress route
- Early commitment signals: fire picks, burn scaling, payoff spell density
- Gear direction: staves and fire-reinforcing caster gear
- Boss ask: live long enough to convert payoff turns into an actual kill sequence

### Cold Control

- Role: secondary safe-control path
- Core fantasy: deny the enemy, slow the fight down, then win on safe terms
- What it should feel like: the safest Sorceress lane, but still precision-dependent
- Early commitment signals: freeze or slow tools, control-heavy early spell rewards
- Gear direction: staves with defensive or control-forward support
- Boss ask: avoid overdrafting safety and failing to close

### Lightning Tempo

- Role: secondary high-volatility path
- Core fantasy: draw, energy, acceleration, and swing turns driven by hand texture
- What it should feel like: the most variance-sensitive caster path in the roster
- Early commitment signals: lightning rewards, energy tempo, draw smoothing
- Gear direction: staves with tempo-forward support
- Boss ask: manage volatility rather than assuming the right turn always appears on time

## Roster-Level Balance Intent

- Amazon and Paladin should read as stable “learnable” classes with room for riskier variants.
- Assassin and Sorceress should be the most sequencing- and draw-sensitive classes.
- Barbarian and Druid should split clearly between sturdy lanes and synergy lanes.
- Necromancer should feel setup-heavy and deliberate, not like generic caster value.
- Every flagship lane should usually clear when committed and drafted well.
- Every secondary lane should have a real path to victory, but not necessarily equal consistency.
- Early picks should narrow the lane by Act II, but not force a dead run if the player has to take one or two stopgaps.
