## Identity

Build an army that snowballs. Summoned creatures persist between turns and deal passive damage. Vines provide sustain, spirits provide buffs. The longer the fight goes, the stronger the board.

## Key Cards

- raven — Basic summon
- spirit_wolf → dire_wolf → grizzly — Escalating summon chain
- carrion_vine, solar_creeper — Healing vines
- heart_of_wolverine — Offensive spirit buff
- spirit_of_barbs — Defensive thorn aura
- oak_sage — Life-boosting spirit

## Synergy Loop

1. Deploy summons early (ravens, spirit wolves)
2. Upgrade to dire wolves and grizzly for board presence
3. Vines provide passive healing to hero and summons
4. Spirit buffs amplify all allied damage/defense
5. Army grows each turn — snowball effect

## Gear Priorities

- **Weapon:** Staves or Maces. Memory runeword (Lum+Io+Sol) for staves: +3 energy, +4 dmg, +5 guard, +6 life

## Boss Strategy

Board presence is the strategy. Boss fights become easier with a full summon board absorbing attacks and dealing passive damage. Set up army before boss room when possible.

## Failure Modes

- AoE enemy attacks clear the board (army wipe)
- Slow to build — vulnerable in early turns before army establishes
- Need enough summon cards to rebuild after board clears
- Can run slightly larger decks (14-18) since army persists
