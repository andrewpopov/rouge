- which skill families each class should emphasize
- what a `3`-slot skill bar should look like for each class
- what the starting skill should accomplish
- what early unlocks should add

## Global Rules

### Slot roles

- `Slot 1`: identity slot
- `Slot 2`: tactical slot
- `Slot 3`: commitment slot

### Blueprint rules

- every class should have a reliable identity button from the first fight
- the second slot should solve a combat problem, not just add more damage
- the third slot should define a real commitment moment or capstone window
- the bar should create context for the deck, not replace the deck

### What skills should not do

- own most card-state complexity
- become the main source of draw/discard manipulation
- become the main source of generated cards
- let the player ignore deck quality

## Amazon

### Primary families

- `Command`
- `Answer`
- `Trigger-Arming`

### Bar shape

- `Slot 1`: precise target command
- `Slot 2`: defensive or tempo answer
- `Slot 3`: ranged payoff window

### Starter bar

- start with a command-forward skill
- the starter should make target selection cleaner and set up mercenary-assisted pressure
- it should feel disciplined, not flashy

### First unlock direction

- add a tactical answer skill
- likely anti-backline, anti-burst, or anti-summon
- this keeps Amazon from becoming pure ranged racing

### Third-slot direction

- add a trigger-arming or commitment skill that creates one strong ranged cash-out window
- this slot should make `Bow Volley`, `Javelin Storm`, or `Passive Tempo` feel different depending on deck support

### What the bar should not do

- it should not become a generic multi-shot artillery bar
- it should not solve every defensive problem by itself

## Assassin

### Primary families

- `State`
- `Trigger-Arming`
- `Conversion`

### Bar shape

- `Slot 1`: shadow or burst state
- `Slot 2`: tactical trigger arming
- `Slot 3`: conversion or cash-out commitment

### Starter bar

- start with a shadow-state or setup skill
- the starter should create the feeling that Assassin prepares the fight before cashing it out

### First unlock direction

- add a trigger-arming skill
- good patterns are next-hit mark cashout, next-trap rider, or next-shadow payoff rider

### Third-slot direction

- add a real conversion or commitment skill
- this slot should turn preparation into one decisive turn without becoming the whole kill plan by itself

### What the bar should not do

- it should not become three burst buttons
- it should not make the deck’s setup cards optional

## Barbarian

### Primary families

- `State`
- `Answer`
- `Commitment`

### Bar shape

- `Slot 1`: mastery or aggression state
- `Slot 2`: stabilize-and-continue answer
- `Slot 3`: committed offensive spike

### Starter bar

- start with a sturdy identity skill
- it should make Barbarian feel proactive and understandable from turn one
- it should reward swinging, but intelligently

### First unlock direction

- add a real answer skill
- likely telegraph respect, anti-burst, or anti-screen breaking
- this is how Barbarian stops being a pure race class

### Third-slot direction

- add a commitment skill
- short warcry surge, mastery overdrive, or frontline spike
- this should be the “now we cash in” button

### What the bar should not do

- it should not be just three attack skills
- it should not let bad deck draws stop mattering

## Druid

### Primary families

- `State`
- `Command`
- `Trigger-Arming`

### Bar shape

- `Slot 1`: elemental or shape attunement
- `Slot 2`: summon or ally command
- `Slot 3`: engine arming commitment

### Starter bar

- start with an attunement-style identity skill
- the starter should tell the player Druid is about choosing a state, not just casting generic spells

### First unlock direction

- add a command skill
- likely summon reinforcement, board stabilization, or ally-support control

### Third-slot direction

- add a trigger-arming or commitment skill
- this slot should let elemental or summon decks hit a meaningful engine turn

### What the bar should not do

- it should not flatten elemental, shifter, and summon identity into one mushy support bar
- it should not make summon setup automatic

## Necromancer

### Primary families

- `Command`
- `Conversion`
- `Recovery`

### Bar shape

- `Slot 1`: summon or corpse command
- `Slot 2`: recovery or board reset
- `Slot 3`: corpse/death conversion commitment

### Starter bar

- start with a command-forward skill
- the starter should make Necromancer feel deliberate and board-aware from turn one

### First unlock direction

- add a recovery skill
- likely board stabilization, minion reinforcement, or party sustain tied to death play

### Third-slot direction

- add a conversion commitment
- this should turn corpses, summons, curses, or attrition into a real payoff window

### What the bar should not do

- it should not become generic caster damage
- it should not let summon lanes ignore the vulnerable setup window entirely

## Paladin

### Primary families

- `State`
- `Answer`
- `Command`

### Bar shape

- `Slot 1`: aura or sanctity state
- `Slot 2`: righteous answer tool
- `Slot 3`: party-command or conviction commitment

### Starter bar

- start with a state skill
- the starter should make Paladin feel protective and offensive at the same time
- it should establish that Paladin leads the party, not just survives

### First unlock direction

- add an answer skill
- likely anti-burst, cleanse, anti-attrition, or anti-fire-pressure

### Third-slot direction

- add a command or commitment skill
- aura-anchored spike, sanctified party push, or offensive judgment window

### What the bar should not do

- it should not become a passive turtle bar
- it should not make defensive aura decks close fights automatically

## Sorceress

### Primary families

- `State`
- `Conversion`
- `Commitment`

### Bar shape

- `Slot 1`: elemental attunement state
- `Slot 2`: elemental conversion tool
- `Slot 3`: explosive spell commitment

### Starter bar

- start with a state-forward spell identity skill
- the starter should establish that Sorceress creates a context for spells instead of merely throwing the biggest nuke

### First unlock direction

- add a conversion skill
- likely turn defense into spell pressure, turn setup into spread, or turn element stacks into a burst window

### Third-slot direction

- add a real commitment skill
- this should be the powerful but timing-sensitive spell spike, not a generic damage cooldown

### What the bar should not do

- it should not be three direct damage buttons
- it should not let spell decks ignore hand sequencing and setup

| Paladin | `State` | `Answer` | `Command` |
| Sorceress | `State` | `Conversion` | `Commitment` |

## Roster-Level Intent

Across the whole roster:

- Amazon and Paladin should have the most readable bars
- Assassin and Sorceress should have the most timing-sensitive bars
- Barbarian and Druid should clearly separate stable play from engine play
- Necromancer should have the most board-aware bar

The final test is simple:

- if the bar alone feels like the whole class, it is too strong
- if the bar creates no context for the deck, it is too weak
- if the bar makes the deck more interesting, it is doing the right job
